# Humblify

A simple way to access your Spotify data with focus on new releases.

![Humblify](https://imgur.com/a/iaqZwAZ)

## Installing

1. Install

   ```sh
   npm install 
   ```

2. Start the development server

   ```sh
   npm start
   ```

## Built With

* [React]
* [Spotify API]
* [Axios]
* [Tailwind CSS]
